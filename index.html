Image stitching

App | Github

A client-side only webapplication to join, stitch or align overlapping images. It can be used to created panoramas or to align images (for example forms which have not been scanned properly).

The main technologies used are Vue and OpenCV.

Features: Surface projection, feature detection, feature matching, bundle adjustement, wave correction, color transfer, seam detection and multiband blending.

The core algorithms are taken from OpenCV (modules Feature2D and Stiching mainly) and have been adjusted where needed. A customized version of OpenCV has than been compiled to wasm. To ensure a non blocking UI, all OpenCV related function run via a webworker.

Pros and cons: As image stiching can be quite memory and cpu intensive, there are some limitations on the amount or sizes of images which can be stitched together, especially on mobile devices. On the plus side, basic stitching functionality is available on almost any devices which can run a browser. There is no need to download a professional stitching software. The application is fully functional offline as well.

Identity management

App | Github

A Vue SPA client using OpenId Connect to get access to Identity (authentication) and API resources" (authorization). It uses the oidc-client javascript library.

One backend this app talks to is an authorization server implemented with Asp.net Core and Identity Server 4.

The technologies mentioned above enable, among others, features like Single Sign-On (SSO), login with external providers and Access Control for APIs.

The purpose of this app is to learn about the involved technologies. There is also a webapi running which provides endpoints to test access to api resources. These endpoints require various kinds of authorization. For example the age of a user or the role of a user. If you are logged in, you can try it out here.
